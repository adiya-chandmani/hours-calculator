# Hours Calculator

A fast, SEO-friendly microtool website built with Next.js App Router and TypeScript. It helps users add or subtract hours and minutes for work schedules, duration math, and daily planning.

## Pages

- /
- /about
- /privacy
- /terms
- /contact
- /robots.txt
- /sitemap.xml

## Local development

```bash
npm install
npm run build
npm run dev
```

## Environment variable

Create a `.env.local` file:

```bash
NEXT_PUBLIC_SITE_URL=https://your-domain.com
```

## Notes

- Static-first structure for speed and easy deployment
- Accessible semantic layout
- Sitemap and robots generated from `NEXT_PUBLIC_SITE_URL`
